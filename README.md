# 🌟 Project Highlights

## 📅 Interactive Academic Calendar

The calendar allows students to navigate across months and select specific dates to view the academic schedule assigned for that day.

### Features

* Monthly navigation
* Current day highlighting
* Date selection
* Academic planning support

---

## 📚 Timetable Management

Students can instantly view class schedules associated with a selected date.

### Includes

* Subject name
* Period number
* Class timings
* Day order information

---

## 🔢 Day Order System

The application supports academic day-order scheduling.

### Supported

* Day Order 1
* Day Order 2
* Day Order 3
* Day Order 4
* Day Order 5
* Day Order 6

Schedules update dynamically according to the selected day order.

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/sudalai775/Calendar-Web.git
```

## Navigate to Project

```bash
cd Calendar-Web
```

## Open in Browser

```bash
open index.html
```

Or run a local server:

```bash
python -m http.server 8000
```

Visit:

```text
http://localhost:8000
```

# 🚀 Deployment
